Политика конфиденциальности приложения MASSCHAT

Дата вступления в силу: 25 марта 2025 г.

Последнее обновление: 23 января 2026 г.

1. Введение

Настоящая Политика конфиденциальности описывает, как Чудаков Роман Алексеевич (далее — "Мы" или "Оператор") собирает, использует, хранит и защищает ваши персональные данные при использовании мобильного приложения MASSCHAT (далее — "Приложение"). Приложение предназначено для управления рекламой на LED-панелях сумок доставщиков.

Мы соблюдаем Федеральный закон РФ № 152-ФЗ "О персональных данных" и требования Apple по защите данных пользователей.

Контакты: Чудаков Роман Алексеевич, email: info@mass-chat.ru

2. Собираемые данные

Приложение собирает:

  • Контактная информация: Имя пользователя, номер телефона (для регистрации и уведомлений).
  • Данные использования: Информация о взаимодействии с приложением (выбор смены, загрузка рекламы) для улучшения функциональности.
  • Данные устройства: Модель устройства, версия ОС, IP-адрес (для работы Bluetooth-синхронизации и онлайн-сервисов).
  • Данные отчетов XLSX: Имя пользователя, номер телефона, дата/статус/продолжительность смен — передаются на сервер для генерации файла.
  • Фото из камеры и галереи: Изображения администратора из камеры устройства или галереи для создания пиксельной рекламы LED-экранов (обрабатываются на сервере).
  • Bluetooth-данные: Информация для поиска и подключения к LED-экранам (UUID устройств, статус соединения).

Мы не собираем геолокацию, контакты из адресной книги, финансовую информацию или рекламный идентификатор (IDFA).

3. Цели использования данных

Данные используются исключительно для:

  • Предоставления услуг приложения (регистрация, синхронизация рекламы с LED-экранами сумок).
  • Обеспечения безопасности аккаунтов и предотвращения мошенничества.
  • Анализа использования приложения для улучшения функциональности (агрегированные данные, без персонализации).
  • Соблюдения юридических обязательств.
  • Обработка данных на backend-сервере MASSCHAT для генерации XLSX-отчетов и синхронизации с LED-панелями.
  • Импорт фото из камеры/галереи для создания пиксельной рекламы LED.

4. Передача данных третьим лицам

Мы не продаем и не сдаем в аренду ваши данные. Доступ имеют:

  • Хостинг-провайдеры и облачные сервисы (для хранения данных) — только с договором защиты данных.
  • Сторонние SDK (Bluetooth-библиотеки) — получают только технические данные устройства для работы функции синхронизации, не передают данные третьим лицам и не используют для рекламы.
  • Государственные органы по требованию закона.

Приложение не осуществляет кросс-приложенный трекинг и не использует рекламные сети.

5. Хранение и безопасность

Данные хранятся на серверах в России не дольше 3 лет после удаления аккаунта или по истечении необходимости. Используются:

  • Шифрование HTTPS/TLS для передачи данных.
  • Защита серверов (firewalls, двухфакторная аутентификация).
  • В случае утечки — уведомление пользователей в течение 72 часов.

6. Права пользователей

Вы можете:

  • Запросить доступ к своим данным (ответ в течение 30 дней).
  • Исправить неточные данные.
  • Удалить аккаунт и данные (выполняется в течение 30 дней).
  • Отозвать согласие на обработку (это может ограничить работу приложения).
  • Подать жалобу в Роскомнадзор.

Приложение не предназначено для детей младше 16 лет. Если мы получим данные ребенка — удалим их.

7. Изменения политики

Мы можем обновлять политику. Новые версии публикуются здесь с датой обновления. Продолжение использования = согласие с изменениями.
